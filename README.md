# 🌽 Database For Machine Learning

Repositori ini berisi dataset yang digunakan untuk pembelajaran dan eksperimen **Machine Learning (ML)** serta **Data Science**.

## 📋 Dataset

| File | Deskripsi |
|------|-----------|
| `cereal_imp.csv` | Dataset karakteristik sereal (nutrisi, produsen, rating) |

## 🎯 Tujuan

Dataset ini digunakan untuk:
- Mempelajari algoritma klasifikasi dan regresi
- Praktik data preprocessing
- Eksperimen feature engineering
- Evaluasi model ML

## 🛠️ Cocok Untuk

- **Klasifikasi** — Memprediksi jenis/tipe sereal
- **Regresi** — Memprediksi rating nutrisi
- **Clustering** — Mengelompokkan sereal berdasarkan karakteristik

## 📄 Lisensi

Dataset bersumber dari publik dan digunakan untuk tujuan edukasi.
